# Mini LSM in Kotlin 

This project is about implementing production ready LSM in kotlin. 
Inspired by [mini-lsm](https://github.com/skyzh/mini-lsm) which is implemented in Rust. 

## How to Use 

### Cli

- Use java 21 

```shell
# build jar 
$ ./gradlew :cli:build

# run cli application 
$ java -jar cli/build/libs/cli.jar
```
